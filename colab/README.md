# Colab Launch Layer

Google Colab entry points for AI Studio Colab.

| Directory | Purpose |
|-----------|---------|
| [notebooks/](notebooks/) | Control panel and workflow notebooks |
| [launch/](launch/) | Bootstrap and service startup scripts |
| [utilities/](utilities/) | Drive mount, GPU checks, path helpers |

The canonical launcher is `AI_Studio_Control_Panel_Colab.ipynb` (to be placed in `notebooks/` in a future step).
