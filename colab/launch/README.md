# Launch Scripts

Bootstrap and service startup scripts for Colab and local environments.

## Planned Scripts (Phase 1+)

| Script | Purpose |
|--------|---------|
| `bootstrap.sh` | Initial environment setup (ComfyUI, nodes, paths) |
| `start_comfyui.sh` | Launch ComfyUI server |
| `start_a1111.sh` | Launch Automatic1111 WebUI |
| `mount_drive.sh` | Google Drive mount and path wiring |

Scripts reference path mappings from `configs/paths/` and node manifests from `configs/nodes/`.
